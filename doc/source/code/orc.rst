Orc Example
===============

Orc is a bad guy in the game.

Creating a new Orc
^^^^^^^^^^^^^^^^^^

Orcs can easily be created. 

>>> from source.orc import Orc
>>> o1 = Orc()
>>> o2 = Orc(distance=120)
>>> o3 = Orc(velocity=10)
>>> o4 = Orc(oType='B')
>>> o5 = Orc(120, 10, 'B')
